Our interactive web mapping application allows you to explore the breathtaking landscapes of New Zealand's forests. New Zealand has a diverse range of forest ecosystems, from the lush green expanses of native bushland to the orderly rows of exotic plantations. Our platform not only allows you to explore the complexities of these diverse forests, but it also includes a curated photo gallery highlighting their beauty. Furthermore, it contains useful information about other land cover features in New Zealand.

Please note that this website was created solely for project purposes, and that all information and data used came from open-access repositories. This simple yet robust web mapping site makes use of the skills, knowledge, and techniques learned in the course 158740 Location Systems: Spatial Databases, Tools, and Applications. Furthermore, it demonstrates the possibility of further expansion, as additional data can be incorporated in the future using the same methodologies demonstrated in the functionality of this website.
